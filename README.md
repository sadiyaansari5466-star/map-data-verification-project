# Map Data Verification Project (Mumbai)

## About this project

I made this project to practice how real map data is checked and verified. The idea was simple take business listings from maps and see if the information is actually correct or not.

## What I did

* Collected around 30+ restaurant listings from Mumbai
* Checked details like address, timings, and business names
* Compared map data with other sources like websites and listings
* Marked data as correct, incorrect, or needs checking

## What I learned

While doing this, I noticed that most data is correct, but some places have wrong timings or missing details. It also depends on where you check — not all sources match.

## Tools used

* Google Maps
* Google Search
* Excel

## Extra work

I also added some Urdu-based validation to show how multilingual data can be handled.

## Why I made this

I wanted to understand how data verification works in real jobs, especially roles like map data analyst or data evaluator.
